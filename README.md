## ✨ HomeWorks是什么项目？

- `HomeWorks`我们在2023/12/26制作开源的项目,项目构思是在校园作业管理麻烦的时候产生的想法,并且开发此项目.
- 以下是功能预览
- [ ] 管理与用户的登录
- [ ] 管理后台
- [ ] 管理作业发布
- [ ] 管理URL链接分享
- [ ] 学生主页UI
- [ ] 学生当前作业与提交文件入口
- [ ] 支持学籍号登录(很长)
- [ ] 用户默认密码`123456`然后修改密码
